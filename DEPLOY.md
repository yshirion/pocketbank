# PocketBank — Raspberry Pi Deployment Guide

## Prerequisites

Install Node.js (v20+) and git on the Pi:

```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs git
node -v   # should print v20.x.x
```

---

## First-Time Setup

```bash
# 1. Clone the repo
git clone https://github.com/yshirion/pocketbank.git
cd pocketbank

# 2. Run the setup script (installs deps, creates .env, migrates DB, builds)
chmod +x setup.sh
./setup.sh

# 3. Start the app
npm start
```

Open `http://<pi-ip>:8080` from any device on the same network.

---

## Updating After a Code Change

```bash
git pull
./setup.sh   # re-installs, re-migrates, re-builds
npm start
```

---

## Auto-Start on Boot (systemd)

Create the service file:

```bash
sudo nano /etc/systemd/system/pocketbank.service
```

Paste this (replace `pi` with your username if different):

```ini
[Unit]
Description=PocketBank
After=network.target

[Service]
Type=simple
User=pi
WorkingDirectory=/home/pi/pocketbank
ExecStart=/usr/bin/npm start
Restart=on-failure

[Install]
WantedBy=multi-user.target
```

Enable and start:

```bash
sudo systemctl daemon-reload
sudo systemctl enable pocketbank
sudo systemctl start pocketbank

# Check status
sudo systemctl status pocketbank
```

The app will now start automatically every time the Pi boots.

## Environment Variables (`server/.env`)

| Variable       | Description                        | Default                  |
|----------------|------------------------------------|--------------------------|
| `PORT`         | Port the server listens on         | `8080`                   |
| `JWT_SECRET`   | Secret for signing JWT tokens      | Auto-generated by setup  |
| `DATABASE_URL` | Path to the SQLite database file   | `file:./dev.db`          |

The `JWT_SECRET` is generated automatically by `setup.sh` — you do not need to set it manually.
